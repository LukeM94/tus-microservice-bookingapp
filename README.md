This repo contains my codebase for my Booking App microservice. I developed a ticket booking system using Java 21 and Spring Boot. The main goal was to build a robust REST API that handles events (like concerts) and the tickets for those events. I used SQLite as the database because it’s lightweight and Gradle as the build tool. The focus was on making sure the API followed proper microservice standards, like using DTOs to hide the database structure and implementing HATEOAS so the API is easy to navigate.
